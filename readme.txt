== Description ==

Energy Analytics is an enterprise-grade WordPress plugin for comprehensive energy usage reporting and analytics. It provides powerful tools for collecting, visualizing, and exporting energy consumption data.

= Key Features =

* **Advanced Custom Fields Integration**: Seamlessly works with ACF to create custom data collection forms.
* **Interactive Dashboards**: Visualize energy data with beautiful, interactive charts powered by Chart.js.
* **Comprehensive Reporting**: Filter, sort, and export your energy data in multiple formats (CSV, PDF).
* **Custom Scripts**: Create and manage JavaScript calculations for energy metrics.
* **REST API**: Access your energy data programmatically through a dedicated REST API.
* **CLI Commands**: Perform bulk operations using WP-CLI commands.
* **Role-Based Access Control**: Fine-grained permissions for viewing reports, managing data, and more.
* **Multilingual Support**: Plugin is translation-ready with built-in support for both English and Persian.

= Use Cases =

* **Energy Management**: Track and analyze energy consumption across different sources.
* **Sustainability Reporting**: Generate comprehensive reports for sustainability initiatives.
* **Cost Analysis**: Identify trends and opportunities for cost savings.
* **Compliance Documentation**: Maintain records for regulatory compliance.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/energy-analytics` directory, or install the plugin through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to 'Energy Analytics' in the admin menu to start using the plugin.
4. Configure your ACF fields in the plugin's settings.

== Frequently Asked Questions ==

= Does this plugin require Advanced Custom Fields? =

Yes, Energy Analytics requires the Advanced Custom Fields (ACF) plugin to function properly. The free version of ACF will work, but the PRO version is recommended for advanced features.

= Can I export my data? =

Yes, Energy Analytics allows you to export your data in CSV and PDF formats. You can export individual reports or bulk export filtered data.

= Is the plugin multilingual? =

Yes, Energy Analytics is translation-ready and includes built-in support for English and Persian languages.

= Can I create custom calculations? =

Yes, the plugin includes a script editor that allows you to create custom JavaScript calculations based on your energy data.
